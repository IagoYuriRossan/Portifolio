# Portfolio (React Native + Web)

Projeto base para um portfólio responsivo usando React Native (Expo), compatível com mobile e web.

Como usar

1. Instale o Expo CLI (se ainda não):

```bash
npm install -g expo-cli
```

2. Instale dependências e rode:

```bash
npm install
npm run dev
```

Edite `App.js` e os componentes em `src/components` para adaptar conteúdo e estilo. Adicione imagens em `assets`.

Sobre reload automático

Se você estiver editando arquivos dentro do OneDrive ou em um sistema onde o watcher nativo não detecta mudanças, use o script `dev` que força polling (recarregamento automático quando salvar):

```bash
npm run dev
```

Isso inicia o servidor web do Expo com polling de arquivos para garantir que alterações apareçam automaticamente no navegador sem reiniciar o processo.
