Por favor coloque sua foto de perfil neste projeto como `profile.jpg`.

Passos:
1. Salve a imagem que você enviou como `profile.jpg`.
2. Coloque o arquivo em `assets/profile.jpg` (este diretório).
3. O servidor web do Expo (ou `npm run dev`) mostrará automaticamente a foto no header.

Observações:
- No web, o arquivo será servido como `/assets/profile.jpg`.
- Em dispositivos móveis, para incluir a imagem embutida no app é necessário adicionar o arquivo ao bundle ou usar um URL remoto; por enquanto o app usa a versão web quando disponível e um avatar remoto como fallback.
